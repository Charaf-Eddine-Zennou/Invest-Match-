# InvestMatch

InvestMatch est une **plateforme web d’investissement** visant à connecter des investisseurs et des entreprises cherchant des financements selon leurs critères et préférences, afin de faciliter la mise en relation et la prise de décision.

---

## Description du projet

Ce projet a été développé dans le cadre d’un stage de **Développeur Fullstack** (Juil. 2025 – Sep. 2025) chez **Detaillia Berlin, Allemagne – À distance**.  

L’objectif principal était de créer un système intelligent de mise en relation entre investisseurs et entreprises, avec un backend sécurisé et un frontend interactif et dynamique.

---

## Fonctionnalités principales

- Gestion sécurisée des **utilisateurs** et des **profils d’entreprises**  
- Algorithme de **matching intelligent** basé sur des pondérations multi-critères :  
  - Secteur  
  - Budget  
  - Localisation  
  - Effectif  
  - Seed  
- **Matrice de similarité adaptative** qui ajuste les recommandations selon les interactions utilisateurs (likes/dislikes)  
- Tableau de bord interactif pour les investisseurs et entreprises  
- Filtres de recherche dynamiques et recommandations personnalisées  
- Suivi du projet via GitHub, avec tests unitaires et bonnes pratiques de code

---

## Technologies utilisées

- **Backend** : NestJS, MySQL, TypeScript  
- **Frontend** : Angular, TypeScript  
- **Outils** : GitHub, Postman, VS Code
